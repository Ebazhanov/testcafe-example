The purpose of this repo is to show how to use TestCafe framework
for writing test for Mobile and Web

## Getting started
npm install

## Run test locally
##### web
$ npm run test

##### mobile
$ npm run test:mobile

Read QR-code by your phone then click on the 'Open link' to run the test
