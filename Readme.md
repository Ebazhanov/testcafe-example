The purpose of this repo is to show how to use TestCafe framework
for writing test for Mobile and Web

## Getting started
npm install

## Run test locally
##### web
$ npm run test

##### mobile
$ npm run test:mobile

scan QR-code from console then click on the 'Open link' in your phone
